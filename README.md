## 项目概述

本项目是一款基于 Python Django 开发的图书管理系统，旨在为用户提供便捷的图书信息管理功能。

## 技术栈

- Python
- Django
- MySQL

## 功能模块

### 用户管理

- 登录
- 注册
- 登出
- 修改密码

### 图书管理

- 添加图书
- 添加作者
- 添加出版社
- 查找出版社信息
- 查看图书信息

### 数据处理

- 密码加密
- 信息唯一性判断

## 系统角色

- 系统管理员：负责维护整个系统运行
- 普通用户：可进行图书查询、借阅等操作

## 运行环境

- Python 3.x
- Django 3.x
- MySQL 8.x
